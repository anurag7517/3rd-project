AI Voice Assistant (Python Project)
• Developed a voice-controlled virtual assistant using Python.
• Implemented speech recognition to detect wake word and user commands.
• Integrated text-to-speech using edge-tts for realistic AI voice output.
• Enabled features like Google search, YouTube song playback, time/date queries.
• Used libraries: speech_recognition, edge-tts, pywhatkit, webbrowser.
